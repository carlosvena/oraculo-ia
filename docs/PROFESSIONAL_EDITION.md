# ORÁCULO IA — Edición Profesional

La **Edición Profesional** (Epic 10) convierte a ORÁCULO IA en un entorno de trabajo offline enfocado en la resolución de problemas corporativos, de programación y optimización didáctica diaria.

---

## 🛠️ Módulos Principales

### 1. Casos Reales
Colección offline de casos prácticos estructurados para 8 categorías principales:
- Bancos
- Empresas
- Contadores
- Docentes
- Programadores
- Marketing
- Recursos Humanos
- Investigación

### 2. Biblioteca de Prompts Profesionales
Biblioteca de **más de 300 prompts de alta calidad** organizados sistemáticamente en las categorías:
- Excel, Correo, Informes, Investigación, Análisis, Programación, Automatización, Productividad, Reuniones, Bancos.
Cada prompt cuenta con objetivo, cuándo usarlo, cómo adaptarlo con variables delimited, errores comunes, ejemplo y resultado esperado.

### 3. Plantillas Corporativas
Formatos markdown listos para copiar con un toque:
- Informe ejecutivo, Resumen, Checklist, Procedimiento, Acta de reunión, Correo profesional, Documento técnico.

### 4. Simulador Interactivo
Entorno de juego de roles donde el usuario analiza solicitudes de gerencia, selecciona el modelo óptimo (ChatGPT, Gemini, Claude), redacta la instrucción y evalúa el cumplimiento de una rúbrica determinando su puntuación final (0-100%).

### 5. Desafíos Prácticos
Catálogo de **más de 200 desafíos** clasificados por nivel de dificultad (Inicial, Intermedio, Avanzado) para afinar habilidades locales.

---

## 📊 Persistencia y Estadísticas
El sistema almacena el progreso offline del usuario en `SharedPreferences` de forma aislada:
- Cantidad de prompts copiados.
- Casos prácticos resueltos.
- Historial y puntaje de simulaciones de gerencia.
- Desafíos resueltos por nivel de dificultad.

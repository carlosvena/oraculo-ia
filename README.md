# ORÁCULO IA

Mentor diario, local-first, para aprender Inteligencia Artificial en español.

Antes de revisar el código, leer [`docs/VISION.md`](docs/VISION.md).

## Estado

Sprint 1: primera experiencia usable y completamente simulada.

## Ejecutar

Requiere Flutter estable con Dart 3.7 o superior.

```bash
flutter pub get
flutter analyze
flutter test
flutter run
```

La arquitectura está en `docs/PLATFORM_ARCHITECTURE.md` y la auditoría más reciente
en `docs/ARCHITECTURE_REVIEW_0_5.md`.
